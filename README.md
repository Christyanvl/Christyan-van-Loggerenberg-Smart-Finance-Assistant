# ============================================================
# HOW TO USE THE SAVINGS GROWTH PLANNER APP
# ============================================================

# ------------------------------------------------------------
# STEP 1 - OPEN THE APP
# ------------------------------------------------------------

# Run all cells in Google Colab
# from top to bottom.

# After running the notebook,
# a Gradio link will appear.

# Open the Gradio link
# to launch the app.

# ------------------------------------------------------------
# STEP 2 - ENTER SAVING GOAL INFORMATION
# ------------------------------------------------------------

# In the "Saving Goal" section, enter:

# - what you are saving for
# - saving goal amount
# - years to reach goal
# - how much you want to save monthly

# Example:
# Saving Goal = House
# Goal Amount = 1000000
# Years To Goal = 5
# Monthly Saving = 20000

# ------------------------------------------------------------
# STEP 3 - ENTER INCOME AND EXPENSES
# ------------------------------------------------------------

# In the "Income and Expenses" section,
# enter:

# - monthly income
# - groceries expense
# - transport expense
# - entertainment expense
# - rent or bills
# - other monthly expenses

# Example:

# Monthly Income = 70000
# Groceries = 8000
# Transport = 4000
# Entertainment = 5000
# Rent or Bills = 15000
# Other Expenses = 7000

# ------------------------------------------------------------
# STEP 4 - ENTER BANK INTEREST INFORMATION
# ------------------------------------------------------------

# In the "Interest and Results" section:

# - enter the bank interest rate
# - choose monthly or yearly results

# Example:
# Interest Rate = 5

# ------------------------------------------------------------
# STEP 5 - CALCULATE SAVINGS PLAN
# ------------------------------------------------------------

# Click the:
# "Calculate Savings Plan"
# button.

# The app will automatically calculate:

# - total monthly expenses
# - available monthly savings
# - interest income
# - savings growth
# - progress toward the goal
# - estimated months needed
#   to reach the goal

# ------------------------------------------------------------
# STEP 6 - VIEW RESULTS
# ------------------------------------------------------------

# The app will display:

# - savings advice
# - savings growth table
# - months needed to reach goal
# - final full savings plan

# ------------------------------------------------------------
# SAVINGS GROWTH TABLE
# ------------------------------------------------------------

# The savings growth table shows:

# - amount saved
# - interest income
# - total savings
# - progress toward the goal

# Users can choose:
# - monthly results
# - or yearly results

# ------------------------------------------------------------
# FINAL FULL SAVINGS PLAN
# ------------------------------------------------------------

# The final results section displays:

# - saving goal
# - goal amount
# - chosen monthly saving
# - estimated final savings
# - bank interest rate
# - savings progress

# ------------------------------------------------------------
# NOTES
# ------------------------------------------------------------

# - number fields only accept numbers
# - text fields only accept letters
# - results are displayed in Mauritian Rupees (Rs)
# - the app helps users understand
#   savings growth and financial planning
